# SOC Blue Team Lab with Splunk

This project is a hands-on SOC Blue Team laboratory focused on log analysis,
threat detection, and incident investigation using Splunk.

## Objectives
- Learn Splunk from scratch
- Ingest and analyze Windows logs
- Simulate controlled attacks
- Apply Blue Team and SOC best practices
- Maintain security and proper documentation

## Environment
- Host OS: Windows 11
- Hypervisor: VMware
- Containerization: Docker
- SIEM: Splunk Enterprise
